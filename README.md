# BookParser 

## Описание проекта.   
Этот проект позволяет парсить книги из открытого доступа. Все книги парсятся в папку **books**.
## Пример использования.   

![](example.gif)  
   
## Подготовка к запуску.  
Установить библиотеки командой.  
```
pip install -r requirements.txt  
``` 
   
**Пример**  
![](token_example.png)   

## Запуск кода.  
```
python3 parser.py
```
